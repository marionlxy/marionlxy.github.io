我回长沙了

发表于 2017-07-17   |  字数统计106   |  阅读时长预计0   |   分类于 生活   |     |   阅读次数

心若没有栖息的地方，到哪里都是流浪。

窗外知了在深深的叫着夏天，但是城市似乎还是缺少了蛙声。

我已回来三月有余，生活够闲，因此满腹牢骚只能付诸文字了，有人说，沉浸读书写字的人，充满对生活的逃避，我想想还是有一定道理的，长沙没有北上广那么拼。

这可以加我为好友！
